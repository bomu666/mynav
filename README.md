# MyNav — 个人资源导航

> 一站式资源导航站，聚合 AI工具 / 编程开发 / 嵌入式 / 动漫追番 / 网盘资源等 92 个常用网站入口

## 在线访问

👉 **[打开导航页](https://bomu666.github.io/mynav/)**

## 功能

- 10 个分类，92 个链接
- 实时搜索（Ctrl+K）
- 收藏系统（localStorage 持久化）
- 访问记录追踪
- 数字键 1-9 快速切换分类
- 暗色主题，响应式设计

## 更新方法

编辑 `index.html` 中的 `LINKS = [` 数组，然后：

```bash
git add -A && git commit -m "更新链接" && git push
```

网站会自动更新。
